About


MBPW3 is my personal site which hosts a number of small internet services I use on a daily basis. Here you can find contact information of common links. Most services here are written in Rust / WASM as I find it much cleaner and faster a solution than JavaScript. This is just a personal hobby so don't expect extreme reliability.


Server 1


HTTPS: 443
Cockit: 9090


This server is used as a firewall / gateway in addition to a Nextcloud instance. It is running Nethserver on a Dell Optiplex 3020 with an added expansion card and is actually quite usable for the price. I also use this with OpenVPN to tunnel home whenever I need to.


Server 2


SSH: 22
MC: 25565


This is my goto server for offloading work or running any script thats need to be kept alive for extended periods of time. Any project thats needs a testing server for anything between database management to a Discord bot uses this server. For compatibility and support this server runs Ubuntu Server.
