MBPW3


Michael Bednarek's Personal World Wide Web


Information


MBPW3 is my personal site which hosts a number of small internet services I use on a daily basis. Here you can find contact information of common links. Everything is self-hosted on a collection of servers repurposed from old hardware. Most services here are written in Rust / WASM as I find it much cleaner and faster a solution than JavaScript. This is just a personal hobby so don't expect extreme reliability.


Prime


SSH: 22

HTTP: 81

This is my goto server for offloading work or running any script thats need to be kept alive for extended periods of time. Any project thats needs a testing server for anything between database management to a Discord bot uses this server, such as this website on Nginx. For compatibility and support this server runs Ubuntu Server.


Meta


HTTPS: 443


This server is used for smaller tasks relating to monitoring and debugging of the network itself. It continuously monitors the other servers checking for open ports and testing ssh connections to ensure all systems are in working condition. It also acts as the router for prime with the help of PFSense.
