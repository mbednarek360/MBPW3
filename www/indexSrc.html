Information

MBPW3 is a collection of services, tools, scripts, and hardware used to reduce my dependency on internet systems from the likes of closed source centralized companies. I prefer to have a much of this data stored on my hardware as possible. These tools run on a collection of servers hosted locally primarily consisting of old laptops which are listed below.


Prime

SSH: 22

This is my goto server for offloading work or running any script thats need to be kept alive for extended periods of time. Any project thats needs a testing server for anything between database management to a Discord bot uses this server.


NAS

SSH: 24
HTTPS: 443

This server is primarily used for file storage and manipulation over SFTP with extension through Nextcloud. I use Nextcloud for password management, note taking, photo backup, and more. This server provides as an easy and reliable way to securely sync data between multiple devices with remote access.


Meta

SSH: 26
HTTP: 80

This server is used for smaller tasks relating to monitoring and debugging of the network itself. It continuously monitors the other servers checking for open ports and testing ssh connections to ensure all systems are in working condition. This site is hosted on this server.
