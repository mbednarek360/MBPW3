MBPW3


Michael Bednarek's Personal World Wide Web


Information


MBPW3 is a collection of services, tools, scripts, and hardware used to reduce my dependency on internet systems from the likes of closed source or centralized companies. I prefer to have a much of this data stored in my control as possible, for both privacy and dependency purposes. These tools run on a collection of mostly inexpensive or obsolete hardware and started as a hobby project.


Prime


SSH: 22
HTTP: 80


This is my goto server for offloading work or running any script thats need to be kept alive for extended periods of time. Any project thats needs a testing server for anything between database management to a Discord bot uses this server. Anything that needs to be port-forwarded or externally connected will also likely use this server, such as this website. For compatibility and support this server runs Ubuntu Server.


Meta


HTTPS: 443


This server is used for smaller tasks relating to monitoring and debugging of the network itself. It continuously monitors the other servers checking for open ports and testing ssh connections to ensure all systems are in working condition. It also acts as the router for these servers with the help of PFSense.
